# Day 8 - LWC Basics

## What is LWC?
Lightning Web Components (LWC) is a modern framework in Salesforce used to build fast and reusable UI components.

## Why Salesforce Uses LWC
Salesforce uses LWC because it is lightweight, fast, reusable, and based on modern JavaScript standards.
